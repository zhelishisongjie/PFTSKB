PFTDP-KB Tutorial

The web framework of the PFTDP-KB has six components, including ‘Home’ page, ‘Browse ’ page, ‘Search’ page,‘visual analysis’ page, ‘contact us’ page and 'Help' page. Functions of PFTDP-KB and usage of each function is as follows:

1.Home

The ‘Home’ page for providing brief introduction of PFTDP-KB.

2.Browse

We provide two different browse options for users to browse the data: classification browse and Key Word browse to search the data collected in PFTDP-KB.

(1) Classification Browse

To browse the data in PFTDP-KB, first, choose the parameters classification/parameters application/application for surgery type/application for period you are interested in (Figure 1a); second, the interface will skip to the information page (Figure 1b) once you click one of the results in the result list (Figure 1c).

(2) Keyword Browse

The database can be searchable using the parameter name, PID,Group Name,Classification,Application,Surgery Type Period of Application and PMID.

3.Search

PFTDP-KB also provides a search tool for fluid therapy. First and foremost, You need to choose parameter infomation( parameter application /classification of parameter/population /BMI)which you want to. And then, choose the surgery information(surgical type/surgical approach/period of fluid therapy/ASA physical status) which you want to. Click”submit”button, then you'll get the results. Finally, Click”toggle tables/ charts” to get a diagram according to the results.

4.Visual Analysis

Visual analysis tools assist users to analyze PFTDP-KB data in five types: publication years, country distribution, sample size and year distribution, decision parameter pie, and decision parameter for surgery type. Take the publication years analysis as an example, firstly, users are required to select data to analyze. Then click the ‘submit’ button to generate and visualize the results.
